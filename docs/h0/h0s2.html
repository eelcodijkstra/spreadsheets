Toetsvragen¶

In sommige lessen kun je toetsvragen tegenkomen. Hiermee kun je zelf controleren of je de behandelde stof begrijpt. Bovendien helpen deze vragen je om de belangrijke punten te onthouden.

Door op “Check me” te klikken kun je je antwoord controleren.

We gebruiken de volgende soorten toetsvragen:

Multiple choice vragen¶

Kies het alternatief dat volgens jou het best past bij de gestelde vraag.

    Q-1: What programming language does this site help you to learn?

  • Python
  • Yes, Python is a great language to learn, whether you are a beginner or an experienced programmer.
  • Java
  • Java is a good object oriented language but it has some details that make it hard for the beginner.
  • C
  • C is an imperative programming language that has been around for a long time, but it is not the one that we use.
  • ML
  • No, ML is a functional programming language. You can use Python to write functional programs as well.

    Q-2: Which colors might be found in a rainbow? (choose all that are correct)

  • red
  • Red is a definitely on of the colors.
  • yellow
  • Yes, yellow is correct.
  • black
  • Remember the acronym...ROY G BIV. B stands for blue.
  • green
  • Yes, green is one of the colors.

Fill in the blanks¶

In dit geval moet je in de invulvelden aangeven wat ingevuld moet worden in de gaten in de tekst, aangegeven door |blank|. De antwoorden moet je in dezelfde volgorde geven als de gaten in de tekst.
